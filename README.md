# snipy
ebay bid sniping thing

Before use be sure to modify ```email```, ```password```, ```item_id```, and ```max_bid``` to appropriate values.

```json
{
    "page_load_timeout": 5,
    "page_load_retries": 3,
    "time_before_bid": 50,
    "email": "example@example.com",
    "password": "ExamplePassword",
    "item_id": 123456789012,
    "max_bid": 123.45
}
```
